# Emprestaê Server

Servidor simples criado usando o json-server para testar o front-end do Emprestaê, disponível em: https://github.com/gabrielmendes98/emprestae-web

Todos os dados são salvos no arquivo db.json.

Como o foco do projeto não era o backend, não coloquei a parte de verificação de chamada de rotas que necessitam autenticação e nem verificação do token, contudo, o front-end está completo, enviando o token e funcionaria perfeitamente em um backend real.

## Instalation

`yarn install`

## Start the app

`yarn start`
